## Overview

### The challenge

- Build a text-based number guessing game (1–100)
- Include two difficulty levels:
  - **Easy** – 10 attempts  
  - **Hard** – 5 attempts  
- Provide feedback after each guess (“Too High”, “Too Low”)
- Track remaining attempts and end the game when they run out
