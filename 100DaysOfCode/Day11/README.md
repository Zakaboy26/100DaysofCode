## Overview

### The challenge

- Build a simple text-based **Blackjack** game using core Python concepts.
- The game must:
  - Deal cards to player and dealer
  - Handle Aces as 11 or 1
  - Allow the player to *Hit* or *Stand*
  - Force the dealer to draw until reaching at least 17
  - Compare results and determine win/lose/draw

This was the **first Capstone project**, combining everything from the earlier days into one larger program.
